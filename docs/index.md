# Trading Bot Utilities Library Documentation

Welcome to the official documentation for the Trading Bot Utilities Library (tbutilslib). This documentation will help you understand how to use the library effectively for algorithmic trading applications.

## Overview

The Trading Bot Utilities Library provides a comprehensive set of tools for building, testing, and deploying algorithmic trading systems. It includes utilities for data handling, model management, API integration, and more.

## Key Features

- **MongoDB Integration**: Store and retrieve trading data efficiently
- **Schema Validation**: Ensure data integrity with Marshmallow schemas
- **RESTful API Support**: Build trading APIs with Flask
- **Logging Utilities**: Comprehensive logging with rotation
- **Error Handling**: Custom exceptions for better error management
- **Command-line Interface**: Perform common tasks from the command line
